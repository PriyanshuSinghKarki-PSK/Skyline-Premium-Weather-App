# Skyline Weather 🌤️

Skyline is a premium, responsive weather forecast application built with HTML, CSS, and Vanilla JavaScript. It features a modern **glassmorphic** interface and dynamic, animated backgrounds that react to live weather conditions.

## ✨ Features

* **Live Weather Data:** Accurate current weather conditions, 8-hour forecasts, and 5-day projections.
* **Animated Weather Backgrounds:** The background dynamically changes to match current conditions (Sunny, Cloudy, Rain, Snow, Storm, Fog, Night/Stars).
* **Interactive Temperature Chart:** Visualizes the weekly temperature trend using Chart.js.
* **Air Quality Index (AQI):** Displays current air pollution data with a color-coded visual ring.
* **Sun & Moon Tracking:** Visual arc for sunrise/sunset times and real-time moon phase calculation.
* **Smart Tips:** Contextual travel and clothing suggestions based on temperature and weather conditions.
* **Search & Voice Input:** Search by city name or use the Web Speech API for voice-activated search.
* **Geolocation:** Automatically fetches weather for your current location.
* **Favorites & History:** Saves recent searches and favorite cities locally for quick access.
* **Customization:** Toggle between Dark and Light modes, and Metric (°C) or Imperial (°F) units.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Glassmorphism design), Vanilla JavaScript (ES6+).
* **APIs:** * [OpenWeatherMap API](https://openweathermap.org/) (Current Weather, Forecast, and Air Pollution).
  * Web Geolocation API.
  * Web Speech API (Voice Search).
* **Libraries:**
  * [Chart.js](https://www.chartjs.org/) (Temperature trend graph).
  * [Font Awesome](https://fontawesome.com/) (Icons).
  * [Google Fonts](https://fonts.google.com/) (Poppins typography).

## 🚀 Getting Started

### Prerequisites
To get live weather data, you will need a free API key from OpenWeatherMap.
1. Go to [OpenWeatherMap](https://openweathermap.org/) and create a free account.
2. Navigate to your API keys and copy your unique key.
